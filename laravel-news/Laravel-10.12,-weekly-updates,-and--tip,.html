Laravel 10.12

A late day tag brings us to Laravel 10.12.0. Here are the highlights:

  • Add job timeout occurred event in #47068
  • Fix singleton creatable, destryoable, only, except combinations in #47098
  • Add error handling and ensure re-enabling of foreign key constraints in #47182
  • Add assertMethodNotAllowed to TestResponse in #47169
  • Add forceCreateQuietly in #47162
  • Add parameters to timezone validation rule in #47171

You may review the full branch diff on GitHub for a complete list of changes.

This version bump and update is automated for subscribers to a Shifty Plan. If you don't have one of those, be sure to bump your constraint and run composer update to get the latest features.
